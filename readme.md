# git-monthly-insights

Lists git contributors for the last 12 months

## Install

```bash
$ npm install --global git-monthly-insights
```

## CLI

```
$ git-monthly-insights --help

  Usage
    $ git-monthly-insights

  Options
    --repo  Your Repo path

  Examples
    $ git-monthly-insights --repo=./
    September 2023 (1):  Ash Hitchcock
```
